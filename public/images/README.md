## Optimisation requise

⚠️ **TOUTES les images doivent être optimisées avant utilisation :**

1. **Compression** : Réduire à < 200 KB par image
2. **Format** : Convertir en WebP pour meilleure performance
3. **Redimensionnement** : 
   - Hero : 1920x1080px
   - Thumbnails : 800x600px
   - Témoignages : 400x400px
4. **Lazy Loading** : Implémenter loading="lazy" sur toutes les images
5. **Alt text** : Décrire chaque image pour SEO et accessibilité

## Outils de compression recommandés
- TinyPNG / TinyJPG
- Squoosh (Google)
- ImageOptim (Mac)
- Convertisseur WebP en ligne
